## Deskripsi Aplikasi

NusaLestari adalah aplikasi edukasi berbasis Android yang berfungsi sebagai direktori informasi flora dan fauna endemik Indonesia. Aplikasi ini bertujuan untuk meningkatkan kesadaran pengguna terhadap keanekaragaman hayati nusantara dengan menyediakan data komprehensif mengenai klasifikasi, deskripsi, asal sebaran, serta status konservasi dari berbagai spesies unik di Indonesia.

## Fitur yang Diimplementasikan

- Katalog Hewan & Tumbuhan: Menampilkan daftar spesies endemik yang terbagi dalam kategori Hewan dan Tumbuhan.
- Detail Informasi: Menyajikan data mendalam setiap spesies, termasuk klasifikasi ilmiah, habitat/sebaran, status perlindungan, dan deskripsi lengkap.
- Pencarian (Search): Fitur untuk mencari hewan atau tumbuhan tertentu berdasarkan nama secara cepat.
- Favorit: Memungkinkan pengguna menyimpan spesies ke daftar favorit menggunakan database lokal (Room) agar dapat diakses secara offline.
- Navigasi Modern: Implementasi Bottom Navigation Bar untuk perpindahan antar menu (Home, Search, Favorit, Identitas) yang lebih intuitif.
